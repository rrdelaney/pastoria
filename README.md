# Pastoria

[Documentation](https://pastoria.org)

Pastoria is full-stack React framework for building data-driven apps with Relay,
powered by Vite.

- 💫 Pre-configured Relay and StyleX
- 📓 Familiar filesystem router
- 🌐 Built-in GraphQL server
- ⏩ React streaming SSR
